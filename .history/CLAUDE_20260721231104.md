## Rules Learned From AI Workflow Drill

1. Forms must use React Hook Form + Zod schema validation — a form built with plain useState (Round 1) produced more code overall and had no structured way to guarantee every field was actually validated before submit.

2. Accessibility (aria-describedby, labeled inputs) must be explicitly requested in the prompt — it does not appear by default. Round 1's form had no accessibility markup at all; Round 2 only included it because the prompt explicitly required it.
